# V1 Roadmap

## Milestone 1: Markdown-First Ingest

- Import local PDFs and Markdown sources.
- Convert PDFs with MinerU.
- Generate canonical paper bundles.
- Build block, figure, equation, table, and citation indexes.

## Milestone 2: Reader Workspace

- Render Markdown structure tree.
- Render block-level navigation.
- Show evidence-linked summary, figure, equation, and note panels.

## Milestone 3: Knowledge Workspace

- Build per-paper claim and concept nodes.
- Add cross-paper comparison records.
- Create a project graph view.

## Milestone 4: Research Memory

- Add session, project, team, and long-term memory scopes.
- Require evidence refs for durable memory writes.

## Milestone 5: Desktop Integration

- Surface `Inbox`, `Library`, `Reader`, `Graph`, and `Workspace`.
- Add bundle repair and re-index actions.
- Add CLI-to-desktop interoperability for background import tasks.
